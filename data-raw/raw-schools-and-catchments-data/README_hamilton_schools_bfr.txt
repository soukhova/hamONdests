Two RData files include the building footprints.

File `hamilton_schools_bfr_2011.RData` includes a simple features object. The source data is:

Building Footprints Region
Producer: DMTI Spatial Inc.
Date published: 2015-09-01 (publication), 2019-09-15 (revision)

Retrieved from Scholars GeoPortal, and available in directory Antonio/data/Building-Footprints

File `hamilton_schools_bfr_2021.RData` includes a simple features object. The source data is OpenStreetMaps (retrieved January 17, 2021).

These two files were created using notebook `Hamilton-Extract-Building-Footprints.Rmd` found in directory Antonio/data/Building-Footprints.
